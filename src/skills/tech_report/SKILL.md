# Technical Report Generator Skill

## Purpose
Transforms raw performance data and analysis into polished, professional technical reports. Designed for investor presentations, client deliverables, and executive briefings. Creates documents that communicate technical achievements in business terms.

## Capabilities

### Report Types
- **Executive Summary Reports** - High-level overview for decision makers
- **Technical Deep Dives** - Detailed analysis for engineering teams
- **Benchmark Reports** - Performance comparison and validation
- **Progress Reports** - Historical performance tracking over time
- **Optimization Reports** - Bottleneck analysis with recommendations

### Document Features
- Professional formatting and layout
- Executive summaries (BLUF - Bottom Line Up Front)
- Integrated charts and visualizations
- Performance metrics tables
- Comparison analysis
- Actionable recommendations
- Technical specifications
- Custom branding support

### Output Formats
- Microsoft Word (.docx)
- PDF (via docx conversion)
- Markdown (for documentation)

## Usage

The skill automatically loads when you:
- Request a professional report or document
- Ask to create investor/client materials
- Need to present technical results
- Want to document performance achievements

## Files

- `report_generator.py` - Core report generation engine
- `templates.py` - Report templates and layouts
- `formatters.py` - Data formatting and presentation utilities

## Example Workflows

1. **Performance Benchmark Report**
   - Import analysis data
   - Generate executive summary
   - Add visualizations
   - Include technical details
   - Export to Word/PDF

2. **Investor Update**
   - Highlight key achievements
   - Show performance improvements
   - Present competitive advantages
   - Include roadmap/recommendations

3. **Client Deliverable**
   - Professional cover page
   - Executive summary
   - Detailed results
   - Recommendations section
   - Appendix with raw data

## Integration

Works seamlessly with:
- `perf-analyzer` skill for analysis data
- `pptx` skill for presentation slides
- `xlsx` skill for detailed data tables
- `pdf` skill for final distribution

## Customization

Reports can be customized with:
- Company branding (colors, logos)
- Custom templates
- Specific sections/focus areas
- Different technical depth levels
- Target audience adjustments
